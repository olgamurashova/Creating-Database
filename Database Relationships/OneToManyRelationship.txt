1. Creating one-to-many-relationship:

CREATE TABLE page (
  id integer PRIMARY KEY,
  chapter_id integer REFERENCES chapter(id),
  content text,
  header varchar(20),
  footer varchar(20)
);

2. Dropping a column:

ALTER TABLE chapter
DROP COLUMN content;

3. Writing a query to validate the existence of key constraint:

SELECT  
constraint_name, table_name, column_name
FROM 
information_schema.key_column_usage
WHERE 
table_name = 'page';
